## ✨ ฟีเจอร์

### 📋 จัดการงาน
- ➕ เพิ่มการบ้านพร้อมระบุ **ชื่องาน / วิชา / กำหนดส่ง (วันที่และเวลา)**
- 🟡🟠🔴 เลือก **ระดับความสำคัญ** ได้ 3 ระดับ (ชิล ๆ / ปกติ / ด่วนมาก)
- ✅ ติ๊กเสร็จ พร้อม **เอฟเฟกต์ Confetti ระเบิด** ตรงจุดที่คลิก 🎉
- 🗑️ ลบงานทีละชิ้น หรือเคลียร์งานที่เสร็จแล้วทั้งหมดในคลิกเดียว
- 🔍 **ฟิลเตอร์** — ทั้งหมด / ค้างอยู่ / เสร็จแล้ว / เลยกำหนด
- 📊 งานเรียงตามเดดไลน์ใกล้สุดโดยอัตโนมัติ

### ⏱️ นับถอยหลังเรียลไทม์
- อัปเดตทุกวินาที (วัน / ชั่วโมง / นาที / วินาที)
- 🚦 **สีขอบการ์ดบอกความเร่งด่วน** — เหลือง → อำพัน → ส้ม → แดง
- 💀 การ์ด**เต้นเตือน** เมื่อเหลือเวลาน้อยกว่า 1 ชั่วโมง หรือเลยกำหนดส่ง

### 📈 สถิติความคืบหน้า
- นับจำนวนงาน ค้างอยู่ / เสร็จแล้ว / เปอร์เซ็นต์ความคืบหน้า
- แถบ Progress แบบ gradient พร้อม**ประกายแสงวิ่ง** (shimmer)
- 💬 ข้อความให้กำลังใจเปลี่ยนตามความคืบหน้า

### 🎨 ดีไซน์ & ลูกเล่น
- 🖤 ธีม **ส้ม-ดำ** สไตล์นีออน Glassmorphism
- ✨ ดาวส้มทองระยิบระยับ 70 ดวง + ลูกบอลแสงเบลอลอยไปมา
- 🕐 นาฬิกาบอกวันเวลาภาษาไทยแบบเรียลไทม์
- 🔔 Toast แจ้งเตือนทุกการกระทำ
- 🎬 อนิเมชันการ์ดเข้า-ออกลื่นไหล ตัวเลขสถิติเด้งป๊อปเมื่อเปลี่ยนค่า
- 📱 Responsive รองรับทั้งมือถือและเดสก์ท็อป
